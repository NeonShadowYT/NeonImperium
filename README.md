# Neon Imperium

**Официальный сайт инди-разработчика** — портал игр Starve Neon, Alpha 01, ГК Адвенчур и других проектов.

Этот репозиторий содержит исходный код сайта, который служит витриной для игр, точкой распространения (landing page) и центром сообщества. Сайт выполнен как одностраничное приложение (SPA) с поддержкой двух языков (RU/EN) и интерактивными 3D-эффектами.

**[➡️ Посетить сайт](https://neonshadowyt.github.io/NeonImperium)**

## Возможности

*   **Каталог игр**: Главная страница и отдельные страницы для каждого проекта с трейлерами, описаниями, системными требованиями и историей обновлений.
*   **Многоязычность**: Полная поддержка русского и английского языков с переключением без перезагрузки страницы (реализовано на чистом JavaScript).
*   **Интерактивность**:
    *   3D-эффект наклона (tilt) для карточек проектов.
    *   Параллакс для фоновых изображений в шапках страниц.
*   **Адаптивный дизайн**: Корректное отображение на всех устройствах — от мобильных телефонов до широкоформатных мониторов.
*   **Центр сообщества**: Разделы с разработчиками и ютуберами, ссылки на Discord и Telegram.
*   **Лицензионное соглашение**: Отдельная страница с подробными условиями использования игр.

## Стек технологий

*   **HTML5** — семантическая вёрстка.
*   **CSS3** — кастомные свойства (переменные), градиенты, flexbox/grid, медиа-запросы.
*   **JavaScript (Vanilla)** — переключение языка, интерактивные эффекты (без фреймворков).
*   **Шрифты**: Google Fonts (`Russo One`).
*   **Иконки**: Font Awesome 6 (через CDN).
*   **Видео**: Встраивание YouTube-плейлистов.

## Быстрый старт

Чтобы запустить сайт локально, выполните следующие шаги:

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/NeonShadowYT/neon-imperium.git
    cd neon-imperium
    ```

2.  **Откройте в браузере**
    Самый простой способ — открыть файл `index.html`. Для этого можно просто дважды кликнуть по нему в файловом менеджере.

    Для более удобной разработки (с автоматическим обновлением при изменениях) рекомендуется использовать простой локальный сервер. Например, если у вас установлен Python:
    ```bash
    # Python 3
    python -m http.server
    ```
    Затем откройте `http://localhost:8000` в браузере.

3.  **Измените под себя**
    Весь код находится в корневой папке. Основные файлы для кастомизации:
    *   `index.html`, `starve-neon.html` и другие HTML-страницы.
    *   Папка `css/` — все стили разделены по компонентам.
    *   `lang.js` — файл с переводами.
    *   `effects.js` — скрипты для анимаций.
